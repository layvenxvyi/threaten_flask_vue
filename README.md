# threaten_flask_vue
爬取外部威胁漏洞情报数据做展示并做微信推送，可自己加爬威胁漏洞情报源，前后端分离，前端vue，后台使用py3-flask

# 威胁情报
-   重构原本的原生前端为vue，可以简单实现翻页功能，并实现前后端分离
-   前端使用vue框架
-   后台使用python3的flask框架
-   目前威胁情报源只有CNNVD，腾云云，阿里云，绿盟，可自行添加爬取的漏洞情报源，处理主流程不变
